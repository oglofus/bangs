Oglofus Bangs

Oglofus Bangs provides lightning-fast search redirects using bangs. Use shortcuts like "filter bubble !w" to search directly on other sites.

About Oglofus Bangs

Oglofus Bangs is a high-performance, open-source alternative to DuckDuckGo's bangs. It provides lightning-fast redirections using hashed binary search for bang commands.

Examples: filter bubble !w - Search Wikipedia for "filter bubble" mechanical keyboard !gh - Search GitHub for "mechanical keyboard" python book !a - Search Amazon for "python book"

Set as Default Search Engine

Chrome

Settings → Search engine → Manage search engines → Add
Enter "Oglofus Bangs" as name, "search" as keyword, and the URL above

Firefox

Right-click the search box → Add a Keyword for this Search → Enter details
To set as default: Settings → Search → Default Search Engine

Edge

Settings → Privacy, search, and services → Address bar and search → Manage search engines
Add → Enter details → Make default

Safari

Safari → Settings → Search → Select "Other..." → Enter URL
